When the user asks for a **draft PR**, they mean **publish the plan in the PR** (docs/plans, backlog, skill outline) — **not** ship implementation code. Implementation follows in a separate PR or after they assign an agent to the plan PR.

Exception: if implementation already started in the same session, finish it when the user says to complete.
